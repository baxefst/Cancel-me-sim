A physics-based social media outrage simulator. 
Powered by Groq + Gemini. Built with D3-force + Canvas.

[Live Demo]                                                                      
1) Download and run in cmd the following commands                              
2) npm install                                                                 
4) npm fund                                                                       
5) npm run dev

 [How it works] 
 1) In the homepage of the website in the box write down what you want to post   
 2) In the top left corner in settings click on it and upload your Groq and Google api keys which are stored locally in your system
 3) The nodes will be spwaned and discuss among themselves acting as internet users who will react to your statement acting as real internet
 4) After 6 rounds the report will be give how much backlash will it get and by which kind of people and should you post it or not.
